ACCESS HAWAII COMMITTEE

Hawaii Information Consortium General Manager’s Report

March 2015 - May 2015

Executive Summary

Welcome to new Access Hawaii Committee Members.

Over the last three months we have been busy supporting multiple services.  We have 25 services in the design, development or testing phases.  Since the deployment of the new Vital Records system, we have been making tweaks and alterations to the system to improve throughput and efficiency.  The final updates on Modernized eFiling were completed and that service has already processed over 465,000 personal income tax filings.  We also deployed our first service for the Hawaii State Ethics Commission.  The DCCA BReg Division myBusinessAlerts service won the International Association of Commerce Administrators (IACA) Merit Award and my.hawaii.gov was recognized as a Harvard Ash Center Bright Idea in Government.

We are extremely excited to be working in our 15th year as Hawaii's Internet Portal Provider and look forward to continuing to work with our county and state partners for three more years with a contract renewal this year.  We have gathered reference letters from many of our partners which are included in the meeting packet and online.

Highlights


HIePRO Functionality Upgrades

The State Procurement Office electronic procurement system (HIePRO) continued to post system upgrades, including the integration of the MyHawaii and the HIC Notification system. The integration of these modules into HIePRO allow users to see their cost savings and their history of messages received from the system.

HTSB Licensing System - Emergency Hires

Emergency hire applications are now able to be submitted by system users, and not just system administrators. This upgrade adds flexibility to the system and allows greater visibility into submissions. 

Modernized Electronic Tax Filings (MEF)

Modernized Electronic Filings (MEF) continued to increase during the 2015 tax filing season. Accepted Returns for January 1-April 30, 2015 increased 4.8% over the same period last year and Rejected Returns decreased 31%. The Number of Payments for January 1-April 30, 2015 increased 21.6% over the same period last year and Payment Totals for January 1-April 30, 2015 increased 262%.  The MEF system processed over 450,000 returns for that period and has already saved taxpayers $676,000 this year.

Mobile Sex Offender App Released

Key upgrades to the Attorney General HCJDC Mobile Sex Offender App were released, including a new framework, mobile alerts, and an updated icon. The mobile app was made available on the Google Play Store (https://play.google.com/store/apps/details?id=com.ehawaii.sex_offender_search&hl=en) and the iTunes store (https://itunes.apple.com/us/app/hawaii-sex-offender-search/id789745370?mt=8). 

Nurses Continue to Renew Online 

The 2015 renewal cycle for approximately 25,000 is in progress as of May 21, 2015.  High adoption rates continue to be seen as online renewals via the MyPVL service.

Ethics Disclosures Online

The first of many HSEC forms, the D-103A, is now available to submit online.  Previously Ethics forms were available as PDF and could only be submitted electronically via Internet Explorer and Outlook.  Now all modern browsers and operating systems are supported.  If its time for you to submit, visit http://m.hi.gov/ethics

Appendix C: Web Analytics Terms

 

  • Hit - A request for a file from the web server. Available only in log analysis. The number of hits received by a website is frequently cited to assert its popularity, but this number is extremely misleading and dramatically over-estimates popularity. A single web-page typically consists of multiple (often dozens) of discreet files, each of which is counted as a hit as the page is downloaded, so the number of hits is really an arbitrary number more reflective of the complexity of individual pages on the website than the website's actual popularity. The total number of visitors or page views provides a more realistic and accurate assessment of popularity.
  • Page View - A request for a file whose type is defined as a page in log analysis. An occurrence of the script being run in page tagging. In log analysis, a single page view generates multiple hits as all the resources required to view the page (images, .js and .css files) are also requested from the web server.
  • Visit / Session - A series of requests from the same uniquely identified client with a set timeout. A visit is expected to contain multiple hits (in log analysis) and page views.
  • Visitor / Unique Visitor - The uniquely identified client generating requests on the web server (log analysis) or viewing pages (page tagging). A visitor can make multiple visits.
